POST /upload

1. Upload with JavaScript (Fetch)

const form = new FormData();
form.append('file', imageFile);

const res = await fetch('https://arsyhost.pages.dev//upload', {
    method: 'POST',
    body: form
});
const data = await res.json();
console.log(data.url);

2. Upload with Python

import requests
files = {'file': open('image.jpg', 'rb')}
r = requests.post('https://arsyhost.pages.dev//upload', files=files)
print(r.json()['url'])

3. Response Format

{
  "success": true,
  "url": "https://arsyhost.pages.dev//file/AgACAgUAA...",
  "name": "photo.jpg"
}
